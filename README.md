<h2>How to use</h2>
<p>

```

- Clone o repositório:
$ git clone git@github.com:RegisSalesRA/flask_flutter.git

- Entre no diretório:
$ cd client

- Instale as dependências:
$ flutter pub get / flutter packages get

- Execute:
$ flutter run



- Start the server
$ cd server

- Execute:
$ python main.py


```

</p>
